# Gruvbox Dark Hard

A dark Ghostty theme based on Gruvbox Dark Hard with subtle transparency and blur.

## Configuration Details

| Setting | Value | Description |
|---------|-------|-------------|
| `theme` | Gruvbox Dark Hard | Color scheme |
| `font-family` | MonoLisa Variable | Primary font |
| `font-size` | 12 | Font size |
| `font-thicken` | true | Thicker font rendering |
| `background-opacity` | 0.95 | Slight transparency |
| `background-blur-radius` | 20 | Background blur effect |
| `cursor-style` | block | Block cursor |
| `cursor-style-blink` | true | Blinking cursor |
| `cursor-opacity` | 0.8 | Slight cursor transparency |
| `macos-titlebar-style` | tabs | Tab-style title bar |
| `window-padding-x` | 15 | Horizontal padding |
| `window-padding-y` | 10 | Vertical padding |
| `copy-on-select` | clipboard | Auto-copy on selection |
| `bold-is-bright` | true | Bold text uses bright colors |
| `minimum-contrast` | 2 | Minimum contrast ratio |
